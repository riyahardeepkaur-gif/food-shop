# 🍔 Food Shop Website

A modern and responsive Food Shop website built using HTML, CSS, and JavaScript. The project provides an attractive user interface for showcasing food items and creating an engaging online food ordering experience.

## 🌐 Live Demo

https://riyahardeepkaur-gif.github.io/food-shop/

## ✨ Features

- Responsive design for desktop and mobile devices
- Attractive food menu layout
- Interactive user interface
- High-quality food images
- Smooth navigation
- Clean and organized code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages (Deployment)

## 📂 Project Structure

```
food-shop/
│
├── index.html
├── restaurant.css
├── script.js
├── images/
└── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/riyahardeepkaur-gif/food-shop.git
```

2. Open the project folder

3. Run `index.html` in your browser
